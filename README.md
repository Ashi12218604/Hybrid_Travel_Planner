# Hybrid Travel Recommendation System

This project uses Machine Learning, NLP, and a hybrid scoring model 
to recommend travel destinations based on user preferences, demographics, 
past travel history, and destination attributes.

## Files included
- app.py : Streamlit application
- All CSV/XLSX datasets
- Trained models (.joblib)
- requirements.txt : For Streamlit deployment

## Running Locally
pip install -r requirements.txt
streamlit run app.py
